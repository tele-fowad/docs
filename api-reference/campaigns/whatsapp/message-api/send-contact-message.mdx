---
title: "Send Contact"
description: "This API is used to send contact to the end user."
api: "POST /api/v3/{phone_number_id}/messages"
playground: true
---

#### Request Body Parameters

<ParamField path="phone_number_id" type="string" required>
  Specifies the recipient’s phone number in international format (e.g.,
  “1xxxxxxxxxx”).
</ParamField>
