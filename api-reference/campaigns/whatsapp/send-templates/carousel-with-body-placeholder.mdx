---
title: "Send Carousel Template Message with Body Placeholder"
sidebarTitle: "Carousel Template with Placeholder"
description: "This API is used to send the carousel template message with body
placeholders to the end user."
api: "POST /v3/{phone_number_id}/messages"
playground: true
---

#### Request Body Parameters

<ParamField path="phone_number_id" type="string" required>
  Specifies the recipient’s phone number in international format (e.g.,
  “1xxxxxxxxxx”).
</ParamField>
