#### Request Body Parameters

<ParamField path="wabaid" type="string" required>
  Specifies the wabaid of the business account.
</ParamField>

<ParamField path="message_template_id" type="string" required>
  Specifies the message template id.
</ParamField>

<ParamField path="template_name" type="string" required>
  Specifies the template name
</ParamField>
