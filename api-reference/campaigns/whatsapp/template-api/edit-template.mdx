#### Request Body Parameters

<ParamField path="message_template_id" type="string" required>
  Specifies the message template id.
</ParamField>

If your template status has been set to REJECTED as part of category validation, you
have several options:

1. Edit the template's components.
2. Edit the template's category.
3. Create a new template.
