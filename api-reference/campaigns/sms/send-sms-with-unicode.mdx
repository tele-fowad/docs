---
title: Send SMS with Unicode
description: "Our web app portal supports sending SMS messages with Unicode characters, allowing you to communicate in multiple languages and include special characters."
api: "POST /sms"
playground: true
---

This feature is essential for reaching a global audience, ensuring that your messages are accurately represented regardless of the language or symbols used.

#### Request Body Parameters

<ParamField path="sender" type="string" required>
  Originator/Alphanumeric ID
</ParamField>

<ParamField path="to" type="string" required>
  Recipient's MSISDN
</ParamField>

<ParamField path="text" type="string" required>
  The phone number for the contact.
</ParamField>

<ParamField path="type" type="string" required>
  Route Type. Use **TRANS** for Transactional type. Use **PROMO** for
  Promotional type. Use **OTP** for OTP type.
</ParamField>

<ParamField path="unicode" type="integer">
  Route Type. Use **TRANS** for Transactional type. Use **PROMO** for
  Promotional type. Use **OTP** for OTP type.
</ParamField>

<CodeGroup>

```json SampleRequest.json
{
  "sender": "senderName",
  "to": "91XXXXXXXXXX",
  "text": "Hello World!",
  "type": "TRANS",
  "unicode": 1
}
```
