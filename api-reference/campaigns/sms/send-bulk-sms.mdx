api: "POST /sms"
playground: true
---

This functionality is ideal for marketing campaigns, event notifications, alerts, and any situation where you need to reach multiple recipients simultaneously. With our bulk SMS feature, you can ensure that your message is delivered quickly and efficiently to all intended recipients.

#### Request Body Parameters

<ParamField path="sender" type="string" required>
  Originator/Alphanumeric ID
</ParamField>

<ParamField path="to" type="string" required>
  Recipient's MSISDN
</ParamField>

<ParamField path="text" type="string" required>
  The phone number for the contact.
</ParamField>

<ParamField path="type" type="string" required>
  Route Type. Use **TRANS** for Transactional type. Use **PROMO** for
  Promotional type. Use **OTP** for OTP type.
</ParamField>
