## Sending OTP

To send OTP, make a POST request with the required data. Here is an example in Python:

```python
import requests

url = "http://sms.telepie.com/v1/sms"
headers = {
    "Content-Type": "application/json"
}
params = {
    "api_key": "<your-api-key>"
}
data = {
    "to": [9999999999],
    "message": "This is your OTP: 123456",
    "route": "otp",
    "sender_id": "0123456",
    "template_id": "A5FyX1bMg"
}

response = requests.post(url, json=data, headers=headers, params=params)
print(response.json())
```
