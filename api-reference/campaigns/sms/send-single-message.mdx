---
title: Send Single Message
description: "Our web app portal allows you to send individual SMS messages efficiently and securely."
api: "POST /sms"
playground: true
---

#### Request Body Parameters

<ParamField path="sender" type="string" required>
  Originator/Alphanumeric ID
</ParamField>

<ParamField path="to" type="string" required>
  Recipient's MSISDN
</ParamField>

<ParamField path="text" type="string" required>
  The phone number for the contact.
</ParamField>

<ParamField path="type" type="string" required>
  Route Type. Use **TRANS** for Transactional type. Use **PROMO** for
  Promotional type. Use **OTP** for OTP type.
</ParamField>

<CodeGroup>

  "text": "Hello World!",
  "type": "TRANS"
}
